## 📌 Project Overview

## ⚙️ How It Works

1. User creates tasks with title, deadline, and priority
2. AI analyzes urgency and importance
3. Tasks are ranked smartly
4. User tracks completion progress
5. Productivity insights are generated

## 🚀 Installation Guide

### Clone Repository

git clone https://github.com/your-username/AI-Powered-Task-Management-System.git

cd AI-Powered-Task-Management-System

### Install Dependencies

For Python:

pip install -r requirements.txt

For Node.js:

npm install

### Run Project

For Python:

python app.py

For Node.js:

npm start
